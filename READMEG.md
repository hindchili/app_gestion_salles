 Application gestion des salles

Nom : Hind Chili
 Description

Ce projet est une application en Python pour gérer des salles.
On peut ajouter, modifier, supprimer et chercher des salles.

Les informations des salles sont :

* code
* description
* catégorie
* capacité

Les données sont enregistrées dans MySQL.

---

## Objectif

Le but est de créer une application avec :

* Python
* base de données MySQL
* interface graphique

---
 Structure

Le projet contient plusieurs dossiers :

* Data : connexion base de données
* models : classe Salle
* services : traitement des données
* views : interface graphique
* main.py : lancer le programme

* Ajouter une salle
* Modifier une salle
* Supprimer une salle
* Rechercher une salle
* Voir la liste des salles

---

 Lancer le projet

1. Installer les dépendances
2. Configurer MySQL
3. Lancer :
   python main.py

---
 Conclusion

Ce projet m’a aidé à comprendre Python, MySQL et les interfaces graphiques.
